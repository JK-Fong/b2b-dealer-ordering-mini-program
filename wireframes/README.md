# B2B 经销商订货小程序 - 原型图设计

本目录包含了 B2B 经销商订货小程序的低保真原型图（Wireframe），展示了三个核心页面的UI设计。

## 设计原则

本原型图遵循以下设计原则：

1. **B2B 专业风格**：采用深蓝色作为主色调，体现商务专业性
2. **信息密度优化**：高效展示关键业务数据，减少无效留白
3. **清晰的视觉层级**：通过颜色、字号、间距区分信息重要性
4. **符合小程序规范**：底部导航栏、卡片式布局、标准交互模式

## 原型图列表

### 1. 首页 - 资金流水 (01-home-page.png)

**核心功能**：
- 账户余额展示（大号蓝色数字）
- 冻结金额和授信额度对比展示
- 资金流水记录列表（支持滚动）

**设计亮点**：
- 渐变蓝色顶部区域，提升视觉吸引力
- 白色卡片展示余额信息，突出重点
- 流水记录使用红色（支出）和绿色（收入）区分
- 每条记录显示交易类型、时间、金额和余额

**交互说明**：
- 下拉刷新获取最新数据
- 点击流水记录可查看详情（可选）

---

### 2. 商品页 (02-products-page.png)

**核心功能**：
- 商品搜索（支持商品名称/编码）
- 商品列表展示（带图片）
- 实时库存显示
- 数量调整和加入购物车
- 底部购物车悬浮栏

**设计亮点**：
- **商品图片**：左侧展示商品实物图（酒类产品）
- 清晰的价格展示（大号红色）
- 库存状态用蓝色标签突出
- 起订量提示和数量步进器
- 底部黑色购物车栏，黄色总价醒目
- 橙色结算按钮，引导用户完成购买

**交互说明**：
- 点击商品图片可查看大图（可选）
- +/- 按钮调整数量，自动校验起订量
- "加入购物车"按钮在库存不足时置灰
- 点击底部购物车栏跳转到购物车页面

**商品展示规范**：
- 图片尺寸：正方形，建议 200x200rpx
- 图片内容：清晰的产品实物照片
- 图片背景：纯色或简洁背景，突出产品

---

### 3. 我的页 (03-mine-page.png)

**核心功能**：
- 用户信息展示
- 购物车快捷入口（带数量徽章）
- 代客下单入口（业务员专属）
- 订单列表展示

**设计亮点**：
- 渐变蓝色头部，圆形头像（显示姓氏首字母）
- 白色菜单卡片，清晰的图标和文字
- 购物车图标带红色徽章，显示商品数量
- 订单卡片展示订单号、状态、日期和金额
- 状态标签用颜色区分（蓝色=进行中，灰色=已取消）

**交互说明**：
- 点击购物车跳转到购物车页面
- 点击"代客下单"进入经销商选择页面
- 点击订单卡片查看订单详情
- 点击"退出登录"弹出确认对话框

## 底部导航栏（TabBar）

所有页面共享统一的底部导航栏：

| 图标 | 名称 | 激活状态 |
| :---: | :---: | :---: |
| 🏠 | Home | 蓝色填充 |
| 📦 | Products | 蓝色填充 |
| 👤 | Profile | 蓝色填充 |

**设计规范**：
- 未激活状态：灰色图标 + 灰色文字
- 激活状态：蓝色图标 + 蓝色文字
- 背景：白色，带顶部细线分隔

## 组件规范

### 卡片（Card）
- 背景：白色
- 圆角：12rpx
- 阴影：0 2rpx 10rpx rgba(0,0,0,0.05)
- 内边距：20-30rpx

### 按钮（Button）
- 高度：60-80rpx
- 圆角：8rpx
- 主按钮：实色背景 + 白色文字
- 次按钮：浅色背景 + 主色文字 + 1px边框

### 列表项（List Item）
- 高度：自适应，最小 100rpx
- 底部分隔线：1px #F0F0F0
- 内边距：20rpx 左右

### 徽章（Badge）
- 背景：#FF5722（红色）
- 圆角：50%（圆形）或 20rpx（胶囊形）
- 字号：20rpx
- 颜色：白色

## 响应式设计

本原型图基于标准小程序屏幕尺寸（375px 宽度）设计，支持以下设备：

- iPhone 6/7/8: 375 x 667
- iPhone X/11/12: 375 x 812
- Android 主流设备: 360-414px 宽度

**适配原则**：
- 使用 rpx 单位实现自适应
- 固定元素（如底部导航栏）高度保持一致
- 可滚动区域自动适配屏幕高度
